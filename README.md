# ТОВ "ТеплоЭнергоСтрой" — корпоративный сайт (PHP 8.4 + MariaDB + Redis)

Полноценный сайт для компании по обслуживанию промышленных холодильных систем и оборудования.

## Что реализовано
- Главная продающая страница с блоками услуг, преимуществ, отзывов и формы заявки.
- Сохранение заявок в MariaDB.
- Кэширование данных (услуги/отзывы) в Redis.
- JSON API: `/api/services`, `/api/reviews`.
- Админ-панель:
  - вход по логину/паролю из `.env`;
  - просмотр заявок;
  - добавление услуг.
- SQL миграции и сиды.
- Docker Compose для запуска всех сервисов.

## Быстрый старт
1. Скопируйте env:
   ```bash
   cp .env.example .env
   ```
2. Установите автозагрузчик:
   ```bash
   composer install
   ```
3. Запустите стек:
   ```bash
   docker compose up -d
   ```
4. Выполните миграции и сиды:
   ```bash
   php bin/migrate.php
   php bin/seed.php
   ```
5. Откройте сайт: `http://localhost:8080`

## Вход в админку
- URL: `/admin`
- Логин и пароль: `ADMIN_EMAIL` и `ADMIN_PASSWORD` из `.env`.

## Структура
- `app/` — MVC-слой, сервисы, репозитории.
- `config/` — конфигурация приложения/БД/Redis.
- `database/` — миграции и seed-данные.
- `public/` — фронт-контроллер и статические файлы.
- `templates/` — шаблоны страниц.
